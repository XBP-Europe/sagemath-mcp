# Asymptotic Expansions[¶](#asymptotic-expansions "Link to this heading")

## The Asymptotic Ring[¶](#the-asymptotic-ring "Link to this heading")

The asymptotic ring, as well as its main documentation is contained in
the module

* [Asymptotic Ring](sage/rings/asymptotic/asymptotic_ring.html).

## Asymptotic Expansion Generators[¶](#asymptotic-expansion-generators "Link to this heading")

Some common asymptotic expansions can be generated in

* [Common Asymptotic Expansions](sage/rings/asymptotic/asymptotic_expansion_generators.html).

## Supplements[¶](#supplements "Link to this heading")

Behind the scenes of working with asymptotic expressions a couple of
additional classes and tools turn up. For instance the growth of each
summand is managed in growth groups, see below.

### Growth Groups[¶](#growth-groups "Link to this heading")

The growth of a summand of an asymptotic expression is managed in

* [(Asymptotic) Growth Groups](sage/rings/asymptotic/growth_group.html) and
* [Cartesian Products of Growth Groups](sage/rings/asymptotic/growth_group_cartesian.html).

### Term Monoids[¶](#term-monoids "Link to this heading")

A summand of an asymptotic expression is basically a term out of the following monoid:

* [(Asymptotic) Term Monoids](sage/rings/asymptotic/term_monoid.html).

### Miscellaneous[¶](#miscellaneous "Link to this heading")

Various useful functions and tools are collected in

* [Asymptotic Expansions — Miscellaneous](sage/rings/asymptotic/misc.html).
