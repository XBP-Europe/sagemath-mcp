# Databases[¶](#databases "Link to this heading")

Sage includes mathematical databases as standard or optional packages,
and provides convenient interfaces for easy access to the databases.

## Standard databases[¶](#standard-databases "Link to this heading")

These databases are available in the standard install of Sage or if Sage can access
Internet.

* [Frank Lübeck’s tables of Conway polynomials over finite fields](sage/databases/conway.html)
* [The On-Line Encyclopedia of Integer Sequences (OEIS)](sage/databases/oeis.html)
* [Local copy of the On-Line Encyclopedia of Integer Sequences](sage/databases/sloane.html)
* [FindStat - the search engine for combinatorial statistics and maps](sage/databases/findstat.html)

## Optional databases[¶](#optional-databases "Link to this heading")

The following databases require you to install optional packages before full access.

* [Cunningham tables](sage/databases/cunningham_tables.html)
* [KnotInfo database](sage/databases/knotinfo_db.html)
* [Cubic Hecke database](sage/databases/cubic_hecke_db.html)
* [Ideals from the Symbolic Data project](sage/databases/symbolic_data.html)
* [Cremona’s tables of elliptic curves](sage/databases/cremona.html)
* [The Stein-Watkins table of elliptic curves](sage/databases/stein_watkins.html)
* [John Jones’s tables of number fields](sage/databases/jones.html)
* [Database of Hilbert polynomials](sage/databases/db_class_polynomials.html)
* [Database of modular polynomials](sage/databases/db_modular_polynomials.html)
* [Database of the zeros of the Riemann zeta function](sage/databases/odlyzko.html)
