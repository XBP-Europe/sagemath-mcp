# Diophantine approximation[¶](#diophantine-approximation "Link to this heading")

The diophantine approximation deals with the approximation of real numbers
(or real vectors) with rational numbers (or rational vectors).
See the article [Wikipedia article Diophantine\_approximation](https://en.wikipedia.org/wiki/Diophantine_approximation) for more information.
