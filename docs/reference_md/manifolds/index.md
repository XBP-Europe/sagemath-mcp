# Manifolds[¶](#manifolds "Link to this heading")

This is the Sage implementation of manifolds resulting from the
[SageManifolds project](https://sagemanifolds.obspm.fr/).
This section describes only the “manifold” part of SageManifolds;
the pure algebraic part is described in the section
[Tensors on free modules of finite rank](../tensor_free_modules/index.html#tensors-on-free-modules "(in Tensors on free modules of finite rank v10.6)").

More documentation (in particular example worksheets) can be found
[here](https://sagemanifolds.obspm.fr/documentation.html).
